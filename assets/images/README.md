This folder contains product images used by the app.

Script: ../scripts/download_images.sh

How to download 15 sample images (royalty-free via Unsplash Source):

  chmod +x ../scripts/download_images.sh
  ../scripts/download_images.sh

Notes:
- Unsplash images are free to use under the Unsplash license. Attribution is appreciated.
- If you prefer specific images, replace the files in this folder with your own images and keep the same filenames (img05.jpg..img19.jpg, picture1.png, img2.png, img3.png, img4.png).
- After adding/changing assets, run `flutter pub get` and rebuild the app.
